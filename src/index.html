Мои работы

My-booking

Учебный проект, выполненный в процессе изучения JavaScript.

Сетка макета адаптивная (responsive). Верстка модульная. Использован препроцессор SCSS. Файлы стилей объединены и минифицированы с помощью Gulp, обработаны в autoprefixer. Использован БЭМ-нейминг и mobile-first.

В процессе JS-кодинга, использовался ООП. Код разделен на 5 модулей. Модули подключены с помощью библиотеки require.js.

Список отелей грузится по AJAX, файлом data.json, расположенным в папке data на сервере. Разметка отелей генерируется автоматически посредством JavaScript из html-шаблона template.

Местами в коде использован JSDoc.

Обо мне

Я начинающий веб-разработчик. Живу в Москве. Ищу возможность получить опыт разработки frontend в реальных проектах.

Уверенно владею HTML и CSS. Верстаю семантично, адаптивно (использую media queries). Активно использую современные возможности CSS3 (flex, animation, transition, transform, filters, gradients). Обладаю навыками модульной верстки с использованием БЭМ-нейминга и препроцессоров (LESS, SCSS). Знаком с методологией mobile-first. Для сборки проектов использую Gulp.

Знаком с основными принципами оптимизации: CSS sprite, SVG sprite, адаптация графики под retina-дисплеи, минификация исходных файлов.

Владею навыками работы с Photoshop и Illustrator на уровне, достаточном для верстальщика. В перспективе, планирую освоить Sketch.

На данном этапе нахожусь в стадии активного изучения JavaScript. Основной упор делаю на изучение нативного JS. В перспективе планирую освоить основные фреймворки JavaScript (React, Vue, Angular).

Имею базовые навыки работы с системой контроля версий Git. В процессе работы приучаю себя к использованию интерфейса командной строки.
